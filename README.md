# 📊 Dashboard de Vendas no Excel

Projeto de análise de vendas desenvolvido no Excel com foco em visualização de dados e tomada de decisão.

## 🚀 Funcionalidades

* 📈 Faturamento por mês
* 📊 Quantidade vendida ao longo do tempo
* 📦 Produtos mais vendidos
* 📉 Margem média mensal
* 🎛️ Filtros interativos por região

## 🎨 Destaques do Dashboard

* Layout profissional com design moderno
* Uso de gráficos variados (linha, barra, coluna)
* Interface limpa e organizada
* Experiência interativa com segmentação de dados

## 🛠️ Ferramentas utilizadas

* Microsoft Excel
* Tabelas Dinâmicas
* Segmentação de Dados
* Gráficos Dinâmicos
